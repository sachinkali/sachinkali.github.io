# Sachin Kumar - Cybersecurity Portfolio 🛡️

Welcome to the repository for my personal portfolio website! This site serves as a central hub for showcasing my professional experience, independent security research, and technical projects.

🌍 **Live Website:** [sachinkali.github.io](https://sachinkali.github.io)

## 📖 Overview

As a Cybersecurity Analyst and Splunk T2 Support Engineer, I built this portfolio to highlight my hands-on approach to solving complex infrastructure challenges. The site features a clean, single-page layout with a responsive, dark-themed "cyber" aesthetic designed for fellow security researchers and tech recruiters.

### Key Sections Included:
*   **About Me:** A brief overview of my background in systems-level programming, digital forensics, and secure network architecture.
*   **Professional Experience:** Details of my work administering and troubleshooting Splunk Enterprise Security (ES) and User Behavior Analytics (UBA).
*   **Independent Research:** Write-ups on my personal lab projects, including stealth isolated architectures, SIEM integration with Suricata/Snort, and cross-platform virtualization.
*   **Technical Skills:** A quick-glance list of my core competencies, tools, and languages.

## ✨ Features

*   **Cybersecurity Aesthetic:** A sleek, minimal dark mode design (`#0a0a0a` background with cyan `#00ffcc` accents).
*   **Terminal Animation:** A custom JavaScript typewriter effect mimicking a Linux root terminal execution upon loading.
*   **Fully Responsive:** CSS Grid layout ensuring perfect readability across desktop, tablet, and mobile devices.
*   **Lightweight:** Built entirely with pure HTML, CSS, and Vanilla JavaScript—no heavy frameworks required.
